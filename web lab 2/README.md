# PromptForge

## Структура 7-1

### Abstracts
- **`_variables.scss`** - змінні (кольори, шрифти, розміри, брейкпойнти)
- **`_functions.scss`** - `rem()`,`color-mix()`
- **`_mixins.scss`** - `container`, `flex-group`, `title-style`, `grid`, `row`, `col`, `col-responsive`
- **`_breakpoints.scss`** - `media-up()`
- **`_animations.scss`** - поява карток

### Base
- **`_reset.scss`** - скидання стилів
- **`_base.scss`** - базові стилі
- **`_typography.scss`** - заголовки, параграфи

### Layout
- **`_container.scss`** - контейнер

### Components
- **`_button.scss`** - стилі кнопок
- **`_header.scss`** - стилі хедера
- **`_footer.scss`** - стилі футера
- **`_product-card.scss`** - картки продуктів
- **`_intro-group.scss`** - секція "Fuel Your AI" та весь інтро в хоум
- **`_prompts-group.scss`** - структура секції з промптами
- **`_catalog-filter.scss`** - фільтри каталогу
- **`_prompt-detail.scss`** - деталі промпту
- **`_cart-items.scss`** - товари в кошику
- **`_order-summary.scss`** - підсумок замовлення
- **`_why-choose.scss`** - секція "Why you should choose PromptForge"

### Utilities
- **`_spacing.scss`** - utility-класи, не використовую
- **`_visibility.scss`** - utility-класи, не використовую

### Themes
- **`_print.scss`** - стилі для друку
